# Telecom Performance Analysis & Forecasting

## Overview
This project analyzes telecom operational data to identify performance trends, forecast revenue, and evaluate churn risk using machine learning techniques. It combines Python-based analytics with an interactive Power BI dashboard to deliver business insights.

---

## Tech Stack

- Python (Pandas, NumPy, Matplotlib)
- Scikit-learn (Linear Regression)
- Power BI (Data Modeling, DAX, Interactive Visuals)

---

## Project Workflow

### 1. Data Simulation & ETL (Python)
- Generated multi-dimensional telecom dataset (Company, Region, Service Type)
- Calculated KPIs: Profit, EBITDA Margin, ARPU, Growth %
- Performed data cleaning and feature engineering
- Conducted grouped aggregations and pivot-based analysis
